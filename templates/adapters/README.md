# Adapter YAML Templates

This directory contains starter YAML templates for adapter configs.

## Available Templates

- `custom_http.yaml` - starter config for custom HTTP integrations
- `mcp.yaml` - starter config for MCP server integrations

## Usage

Copy a template into the runtime adapter directory and customize it:

```bash
cp templates/adapters/mcp.yaml adapters/target_mcp.yaml
vim adapters/target_mcp.yaml
```
